# mps-signal-flow-graphs
Demo MPS project using MPS-extensions diagrams to implement signal flow graphs

# Setup

1. clone the repository.
1. setup required jars:
   ```
   $ ./gradlew  setup
   ```
1. MPS can now be started and the project can be opened.
